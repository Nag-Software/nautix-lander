# Nautix Landing Page

Nautix er en moderne landing page for båteiere som ønsker kontroll over vedlikehold, utstyr, dokumenter og AI-hjelp. Bygget med shadcn/ui, Tailwind 4 og Next.js 15.

![Nautix Landing Page](./public/og-image.jpg)

## Kom i gang

```bash
npm install
```

```bash
npm run dev
```

Åpne [http://localhost:3000](http://localhost:3000) i nettleseren din.

## Teknologi

### Kjerneteknologi

- **Next.js 15** med App Router
- **Tailwind CSS 4** for styling
- **shadcn/ui** komponenter
- **TypeScript** støtte
- **React 19**

### Funksjoner

- **Shadcn UI**: Bruker [shadcn/ui](https://ui.shadcn.com/) kjernekomponenter
- **Tema**: Lys/mørk modus med `next-themes`
- **Skjemaer**: React Hook Form + Zod validering
- **Server Actions**: Next-safe-action integrasjon
- **MDX Support**: For innholdssider
- **Animasjoner**: Motion library integrasjon
- **ESLint/Prettier**: Forhåndskonfigurert kodeformatering
- **Tilpassede fonter**: DM Sans fontfamilie inkludert
- **Ikoner**: Lucide React
- **Responsivt design**: Mobilvenlig layout
- **SEO**: Metadata og OG-bilder inkludert

### Sider

- Hjem/Landing page med Hero, Features, How It Works, Testimonials, CTA og FAQ
- About page
- Pricing page
- FAQ page
- Contact page with form
- Login/Signup pages

### Blocks

- Hero section
- Logo showcase/marquee
- Features section
- Resource allocation section
- Testimonials with carousel
- Pricing table
- FAQ with accordion
- Footer
- Navigation bar

## Deployment

Production-ready and tested for deployment on [Vercel](https://vercel.com)
